## 必要なもの

開発を行うためのPCが必要です。
USBケーブルで開発機と接続するため、USB-Aポートが必要です。
開発環境の構築はプロジェクトのリンク先を確認ください。

## 頒布物

- 開発機 [toms74209200/CO2MO](https://github.com/toms74209200/CO2MO)

- USBケーブル (type-A - type-C)
